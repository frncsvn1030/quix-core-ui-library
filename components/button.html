Button Component

The Button component is used to trigger actions or events.

Base

This is the base button style with default appearance.

Variants

Button variants provide different visual styles for various contexts.

Outline

Outline buttons have a transparent background with a colored border.

Inverted

Inverted buttons are designed for use on dark backgrounds.

Disabled

Disabled buttons indicate that the action is currently unavailable.

Size Modifiers

Buttons can be rendered in different sizes.

Shapes

Button shapes can be customized with different border radius values.
